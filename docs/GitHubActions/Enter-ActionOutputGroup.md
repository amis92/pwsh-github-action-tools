NAME
    Enter-ActionOutputGroup
    
SYNOPSIS
    Begin an output group.
    
    
SYNTAX
    Enter-ActionOutputGroup [-Name] <String> [<CommonParameters>]
    
    
DESCRIPTION
    Output until the next `groupEnd` will be foldable in this group.
    

PARAMETERS
    -Name <String>
        Name of the output group.
        
        Required?                    true
        Position?                    1
        Default value
